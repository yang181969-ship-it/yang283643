关于

这是一个慢慢搭起来，也会继续慢慢长大的个人网站。

我是谁

一个喜欢数学、物理，也喜欢把零散想法整理成结构的人。

我希望把感兴趣的东西一点点留下来，而不是让它们只停在脑子里。

这个网站在记录什么

这里主要放这些内容：

  • Gallery：图片与日常收集
  • Anime：兴趣相关内容
  • Notes：数学、物理与工具笔记
  • Comment：留言与交流

关于这个网站

这个网站对我来说，不只是一个展示页面，更像一个长期维护的个人空间。

我会在这里记录生活、整理笔记，也会把自己学到的东西慢慢放进来，让它一点点变完整。

技术上它主要使用 HTML、CSS、JavaScript 构建，通过 GitHub Pages 和自定义域名部署；笔记部分支持 Markdown 与 KaTeX，评论区接入了 Waline。

我希望它变成什么

希望几年以后再回来看，它已经不只是几个页面，而是一个真正积累起内容、想法和成长过程的地方。

一句话

把喜欢的内容、学过的东西和正在走的路，慢慢放进同一个地方。
